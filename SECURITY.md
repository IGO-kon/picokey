# Security Policy

## Supported versions

Security fixes are applied to the `main` branch.

## Reporting a vulnerability

Please do not open a public issue for security problems.

Use one of these options:

1. GitHub Security Advisories (private report), if enabled in this repository.
2. Contact the maintainer via GitHub: `@IGO-kon`.

Please include:

- A clear description of the issue.
- Reproduction steps.
- Potential impact.
- Any logs or proof-of-concept details.

You should receive an acknowledgment as soon as possible.
